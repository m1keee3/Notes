## Взаимодействие  
### Полив  

- Осуществляется периодически (период задается в конфигурации растения)
	- При слишком высокой влажности полив пропускается
- Количество воды вычисляется используя влажность почвы, размер горшка и конфигурацию растения

### Свет

- В течение светового дня вычисляются световые часы
- В случае их нехватки в конце светового дня включается лампа
	- Лампа работает до конца дня (24:00) или до восполнения световых часов до необходимых для растения

Информация по световым часам за прошедшие дни также сохраняется на сервер

### Увлажнение воздуха

- В случае недостаточной влажности воздуха происходит увлажнение в соответствие с
	1. Объемом горшка
	2. Текущей влажностью
